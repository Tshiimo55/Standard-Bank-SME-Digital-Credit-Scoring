# Standard Bank SME Digital Credit Scoring Platform

A modern web-based financial technology application for assessing creditworthiness of Small and Medium Enterprises (SMEs) with a complete admin dashboard for monitoring applications.

![Standard Bank](images/standard%20bank%20logo.png)

## 🏦 About The Project

This platform digitizes SME lending by providing:
- **Real-time credit assessments** based on business data
- **Digital loan application** interface for customers
- **Risk classification** (Low/Medium/High) with detailed factors
- **Automated decision engine** with custom scoring algorithm
- **Admin dashboard** for monitoring all applications
- **Cloud-based data storage** with Supabase PostgreSQL

## 🚀 Features

### Customer Application (index.html)
- ✅ SME loan application form with validation
- ✅ Real-time credit scoring algorithm
- ✅ Risk classification with visual badges
- ✅ Detailed decision factors explanation
- ✅ Loading indicators and user feedback
- ✅ Empty states and error handling

### Admin Dashboard (admin.html)
- ✅ Overview statistics dashboard
- ✅ Real-time data from Supabase
- ✅ Color-coded risk levels
- ✅ Auto-refresh every 30 seconds
- ✅ Search and filter capabilities
- ✅ Responsive table design

### Frontend

- **PostgreSQL** - Cloud database
