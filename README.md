# Standard Bank SME Digital Credit Scoring Platform - Frontend

A modern web-based financial technology application for assessing creditworthiness of Small and Medium Enterprises (SMEs).

## 🏦 About The Project

This platform digitizes SME lending by providing:
- Real-time credit assessments
- Digital loan applications
- Risk classification (Low/Medium/High)
- Automated decision engine

## 🎨 Design System

- **Primary Color**: Endeavour (#005199)
- **Accent Color**: Pink Salmon (#FF8C9C)
- **Secondary Color**: International Klein Blue (#0032A1)

## 🚀 Features

- SME loan application interface
- Real-time credit scoring
- Risk classification dashboard
- Responsive design
- Modular architecture
